==============================================================
SITE: RadioWaves
==============================================================

ROLE / CONTEXT
--------------
Role: You are Creative Director for an immersive web experience studio.
Context: You are building "RadioWaves" — an interactive radio website where
the user navigates as a first-person POV player inside a 3D realistic room.
Every physical object in the room IS a navigation element. The vintage radio
plays episodes, the rotary phone opens contact, the cork board reveals the
about page, the tape deck unlocks features, the wall calendar shows the
schedule, and the filing cabinet holds the archive. The experience feels like
an escape room meets a lo-fi radio station — atmospheric, gamified, musical.
You have commissioned a talented web designer to build this experience.


BUILD STACK
-----------
1. Stitch (layout generation)
2. UI UX Pro Max (style / palette / typography selection)
3. Firecrawl (reference scrape)
4. Magic / 21st.dev (marketing blocks)
5. Nano Banana 2 (hero + feature imagery)
6. Brand skill (voice + messaging)
7. Design skill (logo, icons, OG image)
8. Design System skill (token architecture)
9. UI Styling skill (shadcn/ui interactive components)


PURPOSE
-------
An interactive radio website where users explore a first-person 3D room to
discover content. Clickable objects in the room navigate to: episodes, about,
features, contact, schedule, and archive. Atmospheric, gamified, and deeply
musical in character.


==============================================================
STEP 2 — BRAND VOICE (for `brand` skill)
==============================================================

Voice archetype: The Midnight DJ — cool, wry, intimate. Speaks directly to
the listener like a late-night radio host who knows exactly what you need to
hear. Confident without being loud. Always on-air.

Tone dimensions (scale 1–5):
- Formal ↔ Casual:        4 (casual, conversational)
- Serious ↔ Playful:      3 (balanced — cool and playful)
- Reserved ↔ Expressive:  4 (expressive)
- Modern ↔ Timeless:      3 (modern with retro soul)

Voice rules:
- Speak in second person ("you," "your frequency") — never corporate.
- Radio/broadcast metaphors are earned, not stacked. One per section max.
- Short punchy sentences. Rhythm matters — copy should feel like it has a beat.
- Warmth through specificity, not exclamation marks.

Words to use:   frequency, tuned in, on-air, signal, static, broadcast,
                dial, transmission, wavelength, live, heard, play
Words to avoid: leverage, synergy, solutions, premium, content, users,
                consumers, engage, platform

Messaging framework:
- Tagline (≤7 words):       "Your frequency. Always on."
- Value prop (≤20 words):   "RadioWaves is a living room full of sound —
                             tune in, explore, and never miss a signal."
- Key phrases (3–5):        "Tune in," "You're on air," "Find your frequency,"
                            "The room is live," "Dial something new"


==============================================================
STEP 3 — DESIGN SYSTEM (for `ui-ux-pro-max` + `design-system`)
==============================================================

Style keywords:   gamified, immersive, lo-fi aesthetic, musical, retro-modern,
                  bold, tactile, cinematic, atmospheric, first-person

Preferred style direction: high-contrast brutalist-retro base with warm
practical lighting overlaid — think a real radio booth at 2am. Bold geometry
cut by warm amber lamp glow. Objects feel physical and clickable. No soft
claymorphism — surfaces are matte, worn, real.

COLOR SYSTEM (semantic roles):
- Primary:         bold yellow #F5E642 (lamp light, hover states, active indicators)
- Secondary:       pure black #000000 (dominant background, deep surfaces)
- Tertiary:        signal red #FF2200 (CTAs, ON-AIR indicators, alerts)
- Accent:          warm amber #FF8C00 (practical light spill, mid-tones)
- Surface-base:    near-black #0d0d0d (room walls, desk)
- Surface-raised:  dark charcoal #1a1a1a (cards, panels)
- Surface-mid:     #2D1554 (purple-tinted surfaces in 3D room)
- Ink:             off-white #f0ece0 (body text on dark)
- Success/info:    muted green #3d7a4a (VU meter active) / amber (info states)

Contrast requirement: WCAG AA minimum on all text. Yellow-on-black and
white-on-black are primary pairings — both exceed AAA.

TYPOGRAPHY:
- Display:   'Syne', sans-serif (bold condensed, broadcast energy)
- Body:      'Syne', sans-serif (clean, 16–18px base)
- Accent:    same as body (consistent brutalist approach)
- Type scale: 1.333 (perfect fourth) modular scale
- Line height: 1.6 body, 1.1 display

SPACING & LAYOUT:
- Spacing scale (4px base): 4, 8, 12, 16, 24, 32, 48, 64, 96, 128
- Container max-width: 1440px (room fills viewport)
- Grid: freeform / absolute positioning within room scene; standard 12-col
  for panel overlays and content drawers
- Section vertical rhythm: panels slide in over the room — no page scroll
  in the core experience

RADII:
- sm 4px  |  md 8px  |  lg 12px  |  xl 20px  |  pill 9999px
Default card radius: md. Buttons: sm (boxy, physical feel).

ELEVATION (practical lighting — no soft clay):
- sm  0 2px 6px   rgba(0,0,0,0.5)
- md  0 6px 20px  rgba(0,0,0,0.6)
- lg  0 16px 48px rgba(0,0,0,0.7)
Hover glow: 0 0 12px rgba(255,220,0,0.35) — yellow lamp spill on active objects.

MOTION LANGUAGE:
- Easing:     cubic-bezier(0.16, 1, 0.3, 1) for panel entrances
- Durations:  micro 100ms, standard 250ms, expressive 500ms, room-load 1200ms
- Stagger:    60ms between list items in open panels
- Hover:      object highlight + yellow glow pulse 200ms
- Panel open: slide up from bottom or fade in from center, 350ms
- Room load:  static noise flash → room fades in, 1200ms total
- prefers-reduced-motion: all transforms collapse to opacity-only at 200ms

- Alt text on every image (descriptive)

ANIMATION ON ASSEMBLY:
- Room load:       static noise flash (80ms) → black → room fade in (1200ms)

- ON-AIR badge:    slow red pulse, 2s loop
